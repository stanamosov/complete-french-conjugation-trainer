# Complete French Conjugation Trainer

**Complete French Conjugation Trainer** is a free Anki deck for practising French verb conjugation through active recall.

It is designed as a real conjugation trainer, not just a reference list. The deck helps learners practise French verb forms through typed answers, tense recognition, example sentences, audio, grammar notes, and visual tense cues.

**Current version:** v0.0.2
**Deck size:** 92 verbs, 7,000+ active-recall cards
**Format:** split into 2 AnkiWeb parts because AnkiWeb shared decks are limited to 250 MB

## How the cards work

Each card asks you to produce the correct French verb form for a given infinitive, pronoun, and tense.

For example, a card may ask for:
```text
aller – je – passé composé
```

The expected answer is:
```text
je suis allé
```

On the back of the card, you can see:

* the correct answer
* the full phrase with the pronoun
* an example sentence
* audio
* verb type, auxiliary, and past participle when useful
* optional grammar notes
* useful external links such as Le Figaro, Reverso, Bescherelle, and Forvo

Forvo links are based on the actual conjugated form, not only the infinitive.

---

## Visual tense cues

Each tense is visually marked with:

* a dedicated color
* a symbol
* a small timeline indicator

The timeline helps show whether a tense belongs mainly to the past, present, or future. These visual cues make it easier to recognize tense families, compare similar forms, and build stronger long-term recall.
